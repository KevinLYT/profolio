# Kevin Li Portfolio

Single-page personal site with multiple saved variants. The repo keeps the current production page (`index.html`), two historical versions for reference, CSS templates, and the image assets used on the site.

## Files
- `index.html` – Magic-style dark layout (current version)
- `profolio_update1.html` – White minimal refresh
- `profolio_initial.html` – Original violet/glassmorphism version
- `templates/` – Standalone CSS mockups (minimal grid, split hero, mono journal)
- `kevin.jpg`, `diao-chat.png` – Photo + DIAO CHAT brand tile
